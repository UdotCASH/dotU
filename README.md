## 🔍 What is .U?

**.U** (or **.u**) is a Top-Level Domain (TLD) on **Unstoppable Domains**, providing decentralized, censorship-resistant domain names for Web3 applications.

### Key Features
- ✅ **True Ownership** - Your domain, your keys, your control
- ✅ **Decentralized** - Built on blockchain, no central authority
- ✅ **Censorship-Resistant** - Cannot be seized or taken down
- ✅ **Web3 Native** - Perfect for dApps, wallets, and crypto services
- ✅ **Public Registration** - Available for anyone to register
- ✅ **IPFS Gateway** - Native wildcard gateway at `*.u.onl`

## 🛠️ Technical Details

### Blockchain
- **Network:** Polygon (MATIC)
- **Standard:** ERC-721 NFT
- **Registry:** Unstoppable Domains Smart Contract

### Resolution
```javascript
// Ethers.js example
const provider = new ethers.providers.JsonRpcProvider('https://polygon-rpc.com');
const resolver = await provider.getResolver('yourname.u');
const address = await resolver.getAddress('yourname.u');
```

### IPFS Integration
```
# Set IPFS Hash (Web3 Usage) for your domain
ipfs://QmYourHash

# Access via u.onl gateway (wildcard support)
https://yourname.u.onl

# Access via public gateway
https://yourname.u.ipfs.dweb.link
```

**U IPFS Gateway (Browser Support):**
- **Gateway:** `u.onl`
- **Wildcard:** `*.u.onl`
- **Example:** `yourname.u.onl` → resolves to your `.u` domain's IPFS content
- **Direct Integration:** Native support for `.u` domains hosted on IPFS

## 🤝 Integration Guide

### For Developers

#### Resolving .U Domains
```javascript
// Using ENS.js
const { default: ENS } = require('@ensdomains/ensjs');

async function resolveUDomain(domain) {
  const ens = new ENS({ provider, ensAddress: '0x...' });
  const resolver = await ens.resolver(domain);
  const address = await resolver.getAddress();
  return address;
}
```
